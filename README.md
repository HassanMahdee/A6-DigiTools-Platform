# DigiTools, Digital Tools Marketplace

A modern single-page application for browsing and purchasing premium digital productivity tools. It is built with React and styled with Tailwind CSS and DaisyUI.

## Technologies Used

- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React
- Vite

## Features

- **Product Browsing**: Browse 8 premium digital tools displayed in a responsive 3-column grid with tag badges and feature lists.
- **Cart Management**: Add and remove products from the cart with real-time count display in the navbar and running total calculation.
- **Toast Notifications**: Get instant feedback on every cart action, including add, remove, and checkout confirmation.
